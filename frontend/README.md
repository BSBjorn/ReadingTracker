# Reading Tracker - Frontend

Vue.js 3 application for the Reading Tracker.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file (see `.env.example`)

3. Start development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## Features

- Dashboard with reading statistics
- Book library management
- Add books manually or search via API (coming soon)
- Track reading sessions
- Visualize reading habits with charts

## Tech Stack

- Vue.js 3 (Composition API)
- Vue Router (routing)
- Pinia (state management)
- Axios (HTTP client)
- Chart.js + vue-chartjs (data visualization)
- Vite (build tool)
